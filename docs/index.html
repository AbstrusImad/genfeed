01 // ABOUT

WHAT IS GENFEED?

Calling an external API from a blockchain is hard. Calling one from a network where multiple validators must independently agree on the result is harder — especially when the data moves every millisecond. GenFeed packages the full set of GenLayer consensus requirements for the entire Binance public market-data API, so you write one line and get a validator-agreed number.

> non-deterministic isolation

Every HTTP request runs exclusively inside a zero-argument closure handed to the GenVM — written inline, right next to the equivalence-principle call. The non-deterministic surface is auditable at a glance.

> per-endpoint consensus strategy

Immutable data (closed candles, symbol metadata) → strict_eq byte-exact. Live data (prices, order book) → every validator independently re-fetches and accepts only within an explicit bps tolerance. No rubber-stamping, ever.

> no floats on-chain

Decimals become atto-scale integers (value × 10¹⁸) via pure string/integer arithmetic. A Python float is never constructed — results are bit-identical on every validator.

> semantic error taxonomy

HTTP 4xx → [EXTERNAL] · 5xx → [TRANSIENT] · bad input → [EXPECTED]. Validator agreement is well-defined even on failure paths.

02 // HOW IT WORKS IN GENLAYER

ONE LINE. FULL CONSENSUS.

Each method runs the web request, drives validator agreement, converts decimals to atto and maps errors — returning a u256, dict or list you store directly.

LEADER fetches

The leader node executes the HTTP call inside the GenVM sandbox and proposes a reading.

VALIDATORS re-fetch

Every validator independently hits the same endpoint and accepts only if both readings agree within your bps tolerance — sign-aware, integer-only math.

STATE agrees

The agreed value lands in your contract storage as an atto integer. Bit-identical on every node. Disagreement rotates the leader — it can never corrupt state.

PYTHON · INTELLIGENT CONTRACT 
# { "Depends": "py-genlayer:<pinned-runner-hash>" }
from genlayer import *
from genfeed import BinanceConnector


class MyOracle(gl.Contract):
    btc_price_atto: u256                      # price × 10**18

    @gl.public.write
    def refresh(self) -> None:
        # one line: HTTP isolation + validator consensus + atto + errors
        self.btc_price_atto = BinanceConnector.get_price("BTCUSDT")

    @gl.public.view
    def price(self) -> str:
        return str(int(self.btc_price_atto))
